# 指南针
一款简单好用的安卓指南针APP

## 开发背景
本人路痴，一加手机又没有官方指南针应用，出门用导航都会走错方向，特别是走地下通道，唉~  寻遍市场上所有指南针应用，发现要么带广告，要么界面太丑，想着也不难，所以花了点时间自己写了一个。
